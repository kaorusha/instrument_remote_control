操作步驟：讀取計數器資料

本說明文件將引導您在 Windows 環境下執行讀取計數器資料的程式。

1. 打開命令提示字元 (cmd)：
   - 按下鍵盤上的 Windows 鍵 (通常是鍵盤左下角有 Windows 標誌的按鍵)。
   - 在開始選單中輸入 "cmd" 或 "命令提示字元"。
   - 點擊 "命令提示字元" 應用程式以開啟。

2. 從預設的 C 槽轉換到 D 槽的 fan_public\My Documents\read_counter_data 目錄：
   - 在開啟的命令提示字元視窗中，首先輸入以下指令切換到 D 槽：
     ```
     d:
     ```
     然後按下 Enter 鍵。命令提示字元應該會顯示 `D:\>`。
   - 接著，輸入以下指令導航到 `fan_public\My Documents\read_counter_data` 目錄：
     ```
     cd "fan_public\My Documents\read_counter_data"
     ```
     按下 Enter 鍵。請注意，如果你的路徑中包含空格，你可能需要用雙引號將路徑括起來。
     
     完成後，命令提示字元應該會顯示 `D:\fan_public\My Documents\read_counter_data>`。

3. 確認序列埠名稱 (COM 埠) 是否正確：
   - 你的程式需要指定正確的序列埠才能與硬體設備通訊。你需要查看裝置管理員以確認你的 USB 轉 RS485 轉換器所使用的 COM 埠號。有幾種方式開啟裝置管理員：
     - **方法一：使用開始選單**
       - 按下鍵盤上的 Windows 鍵。
       - 輸入 "裝置管理員" 並按下 Enter 鍵。
     - **方法二：使用 Windows + X 快速鍵 (推薦)**
       - 同時按下鍵盤上的 **Windows 鍵** (通常是鍵盤左下角有 Windows 標誌的按鍵) 和字母 **X** 鍵。
       - 在彈出的選單中，選擇 "裝置管理員"。
   - 在裝置管理員視窗中，展開 "連接埠 (COM 和 LPT)" 這個類別。
   - 尋找名稱類似 "USB-SERIAL CH340" 或其他與你的 USB 轉 RS485 轉換器相關的裝置。
   - 該裝置旁邊會顯示它被分配到的 COM 埠號，例如 "COM3"、"COM4" 等。
   - **請記下你在裝置管理員中看到的實際 COM 埠號，並在執行程式的命令中替換範例中的預設值。**

4. 執行命令行並輸入參數以聽取多個 Modbus 設備：
   - 在程式所在的目錄下，輸入你的程式的可執行檔名稱 (`read_counter_data.exe`)，後面跟著序列埠名稱，然後是你要聽取的 Modbus 從站 ID 列表，最後是與這些從站 ID 對應的輸出檔案名稱列表。
   - **請將以下 `<實際COM埠號>` 替換為你在裝置管理員中看到的實際 COM 埠號。**
   - **`<從站ID1> <從站ID2> <從站ID3> ...`：替換為你要聽取的 Modbus 設備的實際從站 ID。請用空格分隔多個從站 ID。**
   - **`<檔案名稱1>.csv <檔案名稱2>.csv <檔案名稱3>.csv ...`：替換為要儲存每個從站資料的 CSV 檔案名稱。請確保檔案名稱的順序與從站 ID 的順序對應，並用空格分隔。**

     例如，如果你的程式名稱是 `read_counter_data.exe`，你的 COM 埠是 `COM3`，你要聽取從站 ID 為 `0` 和 `1` 的兩個設備，並將它們的資料分別儲存到 `counter_log_0.csv` 和 `counter_log_1.csv`，則你需要輸入以下指令並按下 Enter 鍵：
     ```
     read_counter_data.exe COM3 0 1 counter_log_0.csv counter_log_1.csv
     ```

     只聽取一個Modbus 從站 ID 是 `0`，儲存檔案名稱是 `counter_log.csv`，則輸入以下指令並按下 Enter 鍵：
     ```
     read_counter_data.exe COM3 0 counter_log.csv
     ```
   - 程式將會開始執行，並定期讀取指定 Modbus 從站的資料，然後將每個從站的資料儲存到對應的 CSV 檔案中。請注意觀察命令提示字元中的輸出訊息，以確認程式是否正常運行。

   - 若要停止程式運行 (如果程式是持續讀取資料)，可以按下 `Ctrl` + `C` 組合鍵。

請確保提供的從站 ID 數量與檔案名稱的數量相同，並且順序對應正確。
